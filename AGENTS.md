# Project Context: TruePace

## 1. Project Overview
TruePace is an AI-powered running trainer that adapts to user feedback. It is a hackathon project.
The goal for the hackathon is to eventually implement Opik to evaluate the AI in this application.
- **Core Loop:** User sets goal -> AI generates plan -> User logs run -> AI adjusts future runs.
- **Current Phase:** Hackathon MVP (Onboarding & Plan Generation).

## 2. Tech Stack & Constraints
- **Framework:** Next.js 15+ (App Router).
- **Language:** TypeScript (Strict).
- **Database:** PostgreSQL (via Supabase).
- **ORM:** Prisma.
- **Styling:** Tailwind CSS + ShadCN UI.
- **Auth:** Supabase Auth.
